# 设计需求：4位ALU

## 功能描述
实现一个4位算术逻辑单元（ALU），支持基本的算术和逻辑运算。

## 接口定义
- 输入：`a[3:0]`、`b[3:0]`（操作数），`op[2:0]`（操作码），`clk`、`rst_n`
- 输出：`result[3:0]`（运算结果），`zero`（结果为0标志），`carry`（进位标志）

## 支持的操作（op编码）
| op | 操作 |
|----|------|
| 000 | ADD: result = a + b |
| 001 | SUB: result = a - b |
| 010 | AND: result = a & b |
| 011 | OR:  result = a \| b |
| 100 | XOR: result = a ^ b |
| 101 | NOT: result = ~a |
| 110 | SHL: result = a << 1 |
| 111 | SHR: result = a >> 1 |

## 时序要求
- 时钟频率：100MHz
- 单周期实现（输入到输出1个时钟周期）
- 异步复位，低有效（rst_n=0时复位）

## 验证要求
- 测试所有8种操作
- 测试边界值（全0、全1）
- 测试进位情况（如 0xF + 0x1）
